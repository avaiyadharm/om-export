[![Live Demo](https://img.shields.io/badge/demo-live-green.svg)](https://om-export.vercel.app)
[![Framework](https://img.shields.io/badge/Framework-Next.js%2016-black.svg)](https://nextjs.org/)
[![Styling](https://img.shields.io/badge/Styling-Tailwind%20CSS%204-38B2AC.svg)](https://tailwindcss.com/)
[![Deployment](https://img.shields.io/badge/Deployment-Vercel-000000.svg)](https://vercel.com/)

**OM Export** is a premium, high-performance web application designed for a global trade and export business. It serves as a comprehensive platform connecting businesses worldwide with premium Indian exports across 50+ countries.

Built with a focus on **visual excellence** and **user engagement**, the platform features smooth animations, a responsive design system, and an extensive product catalog ranging from food products to industrial equipment.

---

## 🚀 Key Features

- **Modern UI/UX**: A state-of-the-art interface designed with a focus on trust, reliability, and global reach.
- **Dynamic Product Catalog**: An expansive database of 200+ products across multiple sectors.
- **Advanced Filtering**: Intuitive category-based filtering and detailed product specifications.
- **Responsive Architecture**: Fully optimized for a seamless experience on mobile, tablet, and desktop devices.
- **Micro-Animations**: Enhanced user interaction through smooth transitions powered by Framer Motion.
- **Direct Inquiries**: Integrated contact and inquiry system for seamless business communication.

## 🛠️ Getting Started

### Prerequisites

- **Node.js**: 18.0 or higher
- **Package Manager**: npm, yarn, or pnpm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/om-export.git
   cd om-export
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Create a `.env.local` file in the root directory:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```
